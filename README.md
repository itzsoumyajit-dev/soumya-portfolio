## 🚀 Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/itzsoumyajit-dev/soumya-portfolio.git
cd soumya-portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up environment variables

Create a `.env` file in the root directory:

```env
VITE_GITHUB_USERNAME=your_github_username
VITE_GITHUB_TOKEN=your_personal_access_token
```

> **How to get a token:** GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained → permissions: `read:user`, `public_repo`

### 4. Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) 🎉

### 5. Build for production

```bash
npm run build
```

## 🌐 Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/itzsoumyajit-dev/soumya-portfolio)

1. Push your code to your GitHub Repository
2. Import the Project into Vercel
3. Add `VITE_GITHUB_USERNAME` and `VITE_GITHUB_TOKEN` to your Environment Variables in Vercel settings
4. Deploy! ✅
