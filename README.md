## `< Hello, World />` &nbsp; I'm Soumyajit Saha

</div>

> 🚀 **[→ Visit the Live Portfolio](https://soumya-portfolio-hoot6p7s2-itzsoumyajit-devs-projects.vercel.app)**

A **cinematic, fully responsive developer portfolio** powered by the **GitHub API** — featuring real-time stats, a Three.js WebGL hero scene, interactive canvas particles, glassmorphism cards, and silky smooth scroll. Built to make a lasting first impression.

## ⚙️ Quick Start

### Step 1 — Clone the repository

```bash
git clone https://github.com/itzsoumyajit-dev/soumya-portfolio.git
cd soumya-portfolio
```

### Step 2 — Install dependencies

```bash
npm install
```

### Step 3 — Configure environment variables

```bash
# Copy the example environment file
cp .env.example .env
```

Then open `.env` and fill in your values:

```env
VITE_GITHUB_USERNAME=your_github_username
VITE_GITHUB_TOKEN=your_personal_access_token
```

> 💡 **How to get a GitHub token:**
> GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens
> Required permissions: `read:user`, `public_repo`

### Step 4 — Start the dev server

```bash
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)** in your browser 🎉

### Step 5 — Build for production

```bash
npm run build
```

The output goes to the `dist/` folder — ready to deploy anywhere!

## 🌐 Deploy to Vercel

<div align="center">

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/itzsoumyajit-dev/soumya-portfolio)

</div>

1. Push your code to GitHub
2. Import the project at [vercel.com](https://vercel.com)
3. Add your **Environment Variables** in Project Settings:
   - `VITE_GITHUB_USERNAME`
   - `VITE_GITHUB_TOKEN`
4. Hit **Deploy** ✅

Or deploy instantly from the CLI:

```bash
npm install -g vercel
vercel --prod
```
