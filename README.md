## 🚀 Live Demo

> **[→ Click here to visit the live site](https://soumya-portfolio-hoot6p7s2-itzsoumyajit-devs-projects.vercel.app)**

A **cinematic, fully responsive developer portfolio** that fetches real-time data from the GitHub API and presents it with a premium, glassmorphism-style UI — complete with 3D scenes, smooth scroll, and a custom cursor.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 **Cinematic Loading Screen** | Smooth progress bar with animated logo that fades into the portfolio |
| 🌊 **Canvas Particle System** | Interactive particles that respond to mouse movement |
| 🪟 **Liquid Glass UI** | Apple-style glassmorphism cards with specular highlights |
| 🌙☀️ **Dark / Light Mode** | One-click theme toggle, persisted to `localStorage` |
| 📊 **Live GitHub Stats** | Stars, forks, followers — fetched fresh from GitHub API on every visit |
| 🃏 **Repo Cards** | Top repositories sorted by stars, with language badges |
| 🥧 **Language Chart** | Interactive Recharts donut chart showing your tech stack breakdown |
| 🖱️ **Custom Cursor** | Smooth magnetic cursor with animated hover ring |
| 🎞️ **Scroll Animations** | `ScrollReveal` component with directional fade-in transitions |
| 🌀 **Smooth Scroll** | Powered by `Lenis` for silky physics-based scrolling |
| 🔮 **3D Hero Scene** | Three.js WebGL background rendered in the hero section |
| 📬 **Contact Section** | Social links + contact button, built with animated cards |
| 📱 **Fully Responsive** | Optimized for mobile, tablet, and desktop |
| ⚡ **Blazing Fast Build** | Vite builds the full app in under 3 seconds |

## ⚙️ Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/itzsoumyajit-dev/soumya-portfolio.git
cd soumya-portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env` file in the root by copying the example:

```bash
cp .env.example .env
```

Then fill in your values:

```env
VITE_GITHUB_USERNAME=your_github_username
VITE_GITHUB_TOKEN=your_personal_access_token
```

> **How to get a GitHub token:**
> GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens
> Required permissions: `read:user`, `public_repo`

### 4. Start the dev server

```bash
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)** in your browser 🎉

### 5. Build for production

```bash
npm run build
```

The output goes to the `dist/` folder and is ready to deploy.

## 🌐 Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/itzsoumyajit-dev/soumya-portfolio)

1. Push your code to GitHub
2. Import the project into [Vercel](https://vercel.com)
3. In **Project Settings → Environment Variables**, add:
   - `VITE_GITHUB_USERNAME`
   - `VITE_GITHUB_TOKEN`
4. Click **Deploy** ✅

Or deploy from the CLI:

```bash
npm install -g vercel
vercel --prod
```
